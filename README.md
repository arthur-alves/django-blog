Instalação:
===========
Requisitos:

>Postgresql  
>pacote python-dev  

```bash
Instaler o PIP:
	- sudo apt-get install python-pip

Instale o virtualenv:
	- sudo apt-get install python-virtualenv

Instale o Virtualenv Wrapper
	-pip install virtualenvwrapper

```
